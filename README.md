# The DZ Camp App

A mobile platform for discovering, organizing, and registering for hiking and camping events. Built with a multi-tenant structure where organizations host events for users.

## Tech Stack

- **Frontend:** React Native (Expo, TypeScript, Expo Router)
- **Backend/Database:** Supabase (PostgreSQL with Row Level Security)
- **Authentication:** Clerk (integrated with Supabase RLS via JWT)

---

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun
- Expo Go app on your physical device (or Android simulator)

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/BOUMESLOUTAbdessamad/dzcamp-android.git](https://github.com/BOUMESLOUTAbdessamad/dzcamp-android.git)
   cd hike-camp-app
   ```
